# 📊 Superstore Sales SQL Analysis

## 📌 Project Overview

This project analyzes the **Sample Superstore Dataset** using **MySQL** to uncover meaningful business insights. The analysis focuses on sales performance, profit trends, customer behavior, product performance, and regional analysis using SQL.

---

## 🎯 Project Objective

The objective of this project is to answer real-world business questions using SQL and help stakeholders make data-driven decisions.

---

## 🛠️ Tools & Technologies

- MySQL Workbench
- SQL
- Sample Superstore Dataset
- Git & GitHub

- What is the total number of orders?
- What is the total sales generated?
- What is the total profit earned?

- Which customers generated the highest profit?
- Which products generated the highest sales?
- What is the monthly sales trend?
- What is the yearly sales trend?

- Which products are least profitable?

## 📈 Key Insights

- Identified the highest-performing product categories based on sales.
- Analyzed regional sales and profit performance.
- Identified top-performing customers.
- Compared sales across different states and cities.
- Evaluated monthly and yearly sales trends.
- Ranked customers and products using SQL Window Functions.
